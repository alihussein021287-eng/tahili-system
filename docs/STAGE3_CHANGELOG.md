# Stage 3 Changelog

تغييرات Stage 3 ركزت على تحسين تجربة الاستخدام اليومي، الأرشفة، التحليلات، الفروع، توثيق جلسات العلاج، البحث الموحد، الاستيراد/التصدير، وجودة البيانات.

## Mobile/PWA

- تحسين تجربة الموبايل للمعالجين والاستقبال.
- تحسين قابلية استخدام القائمة الجانبية والشريط العلوي على الشاشات الصغيرة.
- تحسين عرض الصفحات اليومية مثل الزيارات، الطابور، المهام، المواعيد، وملف المراجع.
- مراجعة manifest وservice worker بدون توسيع التخزين الحساس.

## Official documents archive

- تحسين أرشفة الوثائق الرسمية برقم الكتاب، التاريخ، الجهة، النوع، الموضوع، والربط بالمراجع عند توفره.
- تحسين البحث برقم الكتاب، الموضوع، الجهة، اسم المراجع، والتاريخ.
- تحسين عرض الوثائق داخل ملف المراجع وروابط العرض/الطباعة/المرفق.

## Analytics

- تحسين صفحة الإحصائيات الإدارية بمؤشرات عامة ورسوم واضحة.
- إضافة تجميعات للمراجعين، الزيارات، المواعيد، الجلسات، والصيدلية/المخزون عند توفر الصلاحيات.
- تحسين فلاتر التاريخ والفرع/المركز عند دعم البيانات لذلك.

## Branches

- تحسين إدارة الفروع داخل الإعدادات.
- عرض عدد المستخدمين والمراجعين لكل فرع.
- تحسين ظهور الفرع في المستخدمين والمراجعين وإضافة فلاتر الفرع عند توفر البيانات.
- دعم فلتر الفرع في التقارير والإحصائيات بدون فرض عزل بيانات صارم جديد.

## Therapy session logs

- إضافة جدول `TherapySessionLog` في Prisma لتوثيق نتائج جلسات العلاج.
- ربط السجل بـ `Patient` إلزامياً، و`TherapySession` و`Appointment` اختيارياً.
- إضافة واجهة داخل ملف المراجع لتسجيل التمارين المنفذة، استجابة المراجع، الألم قبل/بعد، الملاحظات، وتوصية الجلسة القادمة.
- عرض آخر سجلات الجلسات داخل تبويب الجلسات.
- إضافة رابط سريع في workload لتسجيل نتيجة جلسة اليوم.
- هذه الميزة تتضمن تغيير قاعدة بيانات، ويجب تطبيق مخطط Prisma عند النشر.

## Unified search

- تحسين `/search` ليعرض نتائج منظمة من المراجعين، الوثائق الرسمية، المواعيد، المهام، الأجهزة، والصيدلية حسب الصلاحيات.
- إضافة تبويبات: الكل، المرضى، الوثائق، المواعيد، المهام، الأجهزة، والصيدلية.
- إضافة فلاتر عامة للنص، التاريخ من/إلى، الفرع، والحالة.
- وضع حدود مناسبة للنتائج لتجنب تحميل الجداول كاملة.
- ربط كل نتيجة بصفحتها التفصيلية المناسبة.

## Bulk import/export

- تحسين استيراد المراجعين عبر CSV مع معاينة قبل الحفظ.
- إظهار عدد السجلات المقروءة، الصحيحة، والخاطئة.
- عرض أخطاء السطور مثل الهاتف غير الصحيح، سنة التولد غير الصحيحة، والتكرار داخل الملف.
- منع التكرار القوي عند الاستيراد باستخدام الاسم + الهاتف + سنة التولد.
- إضافة قالب CSV قابل للتحميل برؤوس واضحة وسطر مثال.
- تحسين تصدير المراجعين حسب الفلاتر الحالية مع دعم العربية في Excel.
- ضبط التصدير الشامل ليحترم صلاحيات المستخدم ولا يضيف أوراقاً لا يملك صلاحية عرضها.

## Data quality

- إضافة صفحة `/patients/data-quality` لفحص جودة بيانات المراجعين.
- كشف المكررات القوية: الاسم + الهاتف + سنة التولد.
- كشف المكررات المحتملة: الاسم + سنة التولد.
- كشف الملفات بدون هاتف، هاتف غير طبيعي، بدون محافظة، بدون نوع إصابة، بدون نوع حالة، والمؤرشفين.
- عرض بطاقات أعداد مختصرة وفلاتر حسب نوع المشكلة.
- عرض روابط مباشرة لملفات المراجعين.
- إضافة تصدير CSV لمؤشرات جودة البيانات بدون تعديل تلقائي للبيانات.

## Deployment note

- لأن Stage 3 أضاف جدول `TherapySessionLog`، أي نشر على بيئة جديدة أو قاعدة لم تستقبل هذا التغيير يحتاج تشغيل `npx prisma db push` أو migration مناسب حسب أسلوب النشر المعتمد.
